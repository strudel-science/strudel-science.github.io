Overview

Design Systems offer rules and guidelines for designing and implementing user interfaces for a product.

Our design system will be framed and guided by Task Flows -- abstract workflows that represent how a user progresses through a UI.

Design systems typically contain elements like:

  • Interface fundamentals like Grid, Theme, Colors, Typography, Layout, etc.
  • Library or a list of components like Button, Inputs etc. which are the building blocks of an interface
  • Collection of common patterns like Login Pages, Search, Filter, etc.
