Task Flows

Task flow examples and starter templates with UX guidelines

STRUDEL Design System is framed and guided by Task Flows, which are a set of steps (represented by a series of screens) that help to accomplish a task and represent how a user progresses through a UI.

Browse the Task Flows

Add Notes

Enables users to add, edit or reply to notes for entities to further annotate, clarify or provide additional context to existing information.

comments
notes

Compare Data

Enables users to scan and compare multiple complex entities with similar information categories to support decision making based on the differences.
